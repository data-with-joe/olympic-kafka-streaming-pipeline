### Olympic Streaming Data Pipeline
## Project Overview

        This project demonstrates an end-to-end streaming data engineering pipeline built using Apache Kafka, Apache Spark Structured Streaming, Delta Lake, and PostgreSQL.

        Olympic athlete event data is streamed into Kafka, processed using the Medallion Architecture (Bronze → Silver), and loaded into PostgreSQL for analytics and visualization.
